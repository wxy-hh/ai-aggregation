# 智能对话页面文件上传功能

## 功能概述

智能对话页面现已支持图片和文件上传功能，用户可以上传多种类型的文件，AI 能够分析文件内容并提供相关建议。

## 支持的文件类型

### 图片文件

- PNG、JPEG、JPG、GIF、WebP
- 直接在聊天界面中显示
- 支持点击放大查看

### 文档文件

- PDF、Word (.doc/.docx)、Excel (.xls/.xlsx)、PowerPoint (.ppt/.pptx)
- 显示为卡片形式，包含文件图标和信息
- 支持下载功能

### 代码文件

- JavaScript (.js)、TypeScript (.ts)
- HTML、CSS、JSON、XML、Markdown
- Python (.py)、纯文本 (.txt)
- 显示为代码文件卡片

## 文件限制

- **单个文件大小**：最大 5MB
- **总文件大小**：最大 50MB
- **文件数量**：最多 10 个文件

## 使用方法

### 上传文件

1. **点击附件按钮**：点击聊天输入框左侧的回形针图标
2. **拖拽上传**：直接将文件拖拽到上传区域
3. **选择文件**：点击上传区域选择本地文件

### 管理附件

- **预览**：图片文件会显示缩略图，点击可放大查看
- **移除**：点击附件右上角的 ✕ 按钮移除文件
- **下载**：点击下载按钮保存文件到本地

### 发送消息

- 可以只发送文件（不输入文字）
- 可以同时发送文字和文件
- 发送前会检查文件上传状态，确保所有文件上传成功

## AI 分析能力

### 图片分析

- 识别图片内容和场景
- 提取图片中的文字信息
- 分析图片的设计和构图
- 提供相关建议和改进意见

### 文档分析

- 提取文档主要内容
- 总结文档要点
- 分析文档结构
- 提供内容优化建议

### 代码分析

- 代码审查和质量检查
- 性能优化建议
- 安全漏洞检测
- 代码风格改进

## 技术实现

### 前端组件

- `FileUpload`：文件上传组件
- `MultimediaMessage`：多媒体消息展示
- `ChatInput`：扩展的聊天输入组件

### API 路由

- `/api/chat/upload`：文件上传接口
- `/api/chat`：扩展的聊天接口，支持多媒体消息

### 数据流程

1. 用户选择文件 → 前端验证
2. 调用豆包文件上传 API → 获取 file_id
3. 构造多媒体消息 → 发送给 AI
4. AI 分析并返回响应

## 错误处理

### 常见错误

- **文件过大**：提示压缩文件或选择较小文件
- **格式不支持**：显示支持的文件格式列表
- **上传失败**：提供重试选项和错误详情
- **网络问题**：自动重试和离线提示

### 解决方案

- 检查文件大小和格式
- 确保网络连接稳定
- 重新选择或压缩文件
- 联系技术支持
