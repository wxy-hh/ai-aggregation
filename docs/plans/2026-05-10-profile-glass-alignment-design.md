# 个人中心玻璃质感对齐设计

## 背景

当前个人中心页面已经具备完整的前端结构，但在移动端与浅色主题下，卡片和字段区块整体偏灰、偏实，视觉上更接近普通浅灰面板，而不是与 `/home` 同系的冷白磨砂玻璃层。用户期望页面继续保持简约高级，同时与全局侧边栏、历史页、首页形成统一系统感。

## 已确认方向

采用方案 A：`/home` 同系增强版。

目标不是单纯把卡片变白，而是重新拉开页面背景、外层卡片、内层字段块三层的亮度、透明度和边缘高光，让卡片在浅色主题下呈现更轻、更透、更有悬浮感的磨玻璃效果。

## 设计要点

1. 页面背景继续使用冷白偏蓝底色，但降低灰度，让内容区和左侧导航的衔接更自然。
2. 外层主卡改为更浅的半透明白玻璃层，增强顶部高光和细描边，减弱当前偏灰的实体感。
3. 内层字段块取消实心白灰底板，改为低对比的乳白玻璃块，保留可读性，同时避免“卡片里再套灰卡片”。
4. 安全退出、危险区域、资源消耗卡片与基本信息卡保持同一玻璃语言，仅在危险区域保留更轻的红色语义。
5. 深色主题延续当前结构，不做大改，只保持与浅色主题一致的层级关系。
