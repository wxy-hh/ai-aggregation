# 渐进式翻译显示功能

## 🎯 功能说明

实现逐句翻译的渐进式显示，提升用户体验：

1. **转录完成后立即显示原文**
2. **每个英文句子显示 loading 效果**
3. **翻译完一句，立即显示一句**
4. **不需要等待全部翻译完成**

## 🔧 实现原理

### 1. 初始化翻译数组

```typescript
// 创建一个全是 null 的数组，表示所有句子都未翻译
const translatedSentences: (string | null)[] = new Array(sentences.length).fill(null);

// 立即设置状态，触发渲染
setTranslationResults(translatedSentences as string[]);
```

**效果**：

- 所有英文句子显示 loading 效果
- 用户知道翻译正在进行

### 2. 逐句翻译并更新

```typescript
for (let i = 0; i < sentences.length; i++) {
  // 翻译当前句子
  const result = await translateText({ text: sentences[i], ... });

  // 更新当前句子的翻译
  translatedSentences[i] = result.translatedText.trim();

  // 立即更新状态，触发重新渲染
  setTranslationResults([...translatedSentences] as string[]);

  // 用户立即看到这一句的翻译结果
}
```

**关键点**：

- 每翻译完一句就调用 `setTranslationResults`
- 使用 `[...translatedSentences]` 创建新数组，触发 React 重新渲染
- 用户看到渐进式的翻译过程

### 3. 显示 Loading 效果

```typescript
// 在 createSegments 中
if (translationTexts[index] === null) {
  translatedText = 'translating'; // 特殊标记
}

// 在 SegmentBlock 中
{segment.translatedText === 'translating' ? (
  <span className="flex items-center gap-2">
    <svg className="w-4 h-4 animate-spin">...</svg>
    <span className="animate-pulse">Translating...</span>
  </span>
) : (
  segment.translatedText
)}
```

**效果**：

- 未翻译的句子显示旋转的加载图标
- 文字有脉冲动画效果
- 视觉上清晰表明正在翻译

## 📊 数据流

### 状态变化流程

```
1. 转录完成
   ↓
2. 初始化翻译数组
   translationResults = [null, null, null, ...]
   ↓
3. 渲染界面
   所有英文句子显示 "🔄 Translating..."
   ↓
4. 翻译第1句
   translationResults = ["Hello", null, null, ...]
   ↓
5. 重新渲染
   第1句显示 "Hello"
   其他句子仍显示 "🔄 Translating..."
   ↓
6. 翻译第2句
   translationResults = ["Hello", "This is good", null, ...]
   ↓
7. 重新渲染
   第1、2句显示翻译结果
   其他句子仍显示 "🔄 Translating..."
   ↓
8. 继续翻译...
   ↓
9. 全部完成
   translationResults = ["Hello", "This is good", "Next is...", ...]
   isTranslating = false
```

### 状态管理

```typescript
// 状态定义
const [translationResults, setTranslationResults] = useState<string[] | null>(null);
const [isTranslating, setIsTranslating] = useState(false);

// 状态值的含义
translationResults = null; // 还未开始翻译
translationResults = [null, null]; // 开始翻译，但都未完成
translationResults = ['Hello', null]; // 第1句完成，第2句未完成
translationResults = ['Hello', 'Hi']; // 全部完成

isTranslating = true; // 翻译进行中
isTranslating = false; // 翻译已完成
```

## 🚀 性能优化

### 当前实现

```typescript
// 每翻译完一句就更新状态
for (let i = 0; i < sentences.length; i++) {
  const result = await translateText(...);
  translatedSentences[i] = result.translatedText;
  setTranslationResults([...translatedSentences]); // 触发重新渲染
}
```

**性能影响**：

- 每句翻译完成触发一次渲染
- 10句话 = 10次渲染
- React 会自动优化，只更新变化的部分
- 性能影响可忽略

### 未来优化（如果需要）

#### 优化1：批量更新

```typescript
// 每翻译完3句更新一次
const batchSize = 3;
for (let i = 0; i < sentences.length; i++) {
  const result = await translateText(...);
  translatedSentences[i] = result.translatedText;

  // 每3句或最后一句时更新
  if ((i + 1) % batchSize === 0 || i === sentences.length - 1) {
    setTranslationResults([...translatedSentences]);
  }
}
```

#### 优化2：使用 useTransition

```typescript
import { useTransition } from 'react';

const [isPending, startTransition] = useTransition();

// 在更新时使用 transition

  setTranslationResults([...translatedSentences]);

```

## 🔍 问题排查

### 如果 Loading 不显示

**检查点1：查看 translationResults 的初始值**

```javascript
console.log('初始翻译结果:', translationResults);
// 应该看到：[null, null, null, ...]
```

**检查点2：查看 segment.translatedText**

```javascript
console.log('Segment 翻译文本:', segment.translatedText);
// 应该看到："translating" 或实际翻译结果
```

**检查点3：查看渲染逻辑**

```typescript
{segment.translatedText === 'translating' ? (
  <span>Loading...</span>  // 应该显示这个
) : (
  segment.translatedText
)}
```

### 如果翻译不是渐进式显示

**检查点1：确认每次都调用了 setTranslationResults**

```typescript
for (let i = 0; i < sentences.length; i++) {
  // ...
  setTranslationResults([...translatedSentences]); // 必须有这行
  console.log('已更新第', i + 1, '句');
}
```

**检查点2：确认使用了新数组**

```typescript
// ❌ 错误：直接修改原数组，React 不会重新渲染
setTranslationResults(translatedSentences);

// ✅ 正确：创建新数组，触发重新渲染
setTranslationResults([...translatedSentences]);
```

**检查点3：查看 React DevTools**

- 打开 React DevTools
- 观察 `translationResults` 状态的变化
- 确认每次翻译完成后状态都更新了

## 💡 扩展功能

### 功能1：显示翻译进度百分比

```typescript
// 计算进度
const translatedCount = translationResults?.filter(t => t !== null).length || 0;
const totalCount = translationResults?.length || 0;
const progress = totalCount > 0 ? (translatedCount / totalCount) * 100 : 0;

// 显示进度
{isTranslating && (
  <span>翻译中... {progress.toFixed(0)}%</span>
)}
```

### 功能2：显示翻译速度

```typescript
const [translationSpeed, setTranslationSpeed] = useState(0);

// 计算速度（句/秒）
const startTime = Date.now();
for (let i = 0; i < sentences.length; i++) {
  // 翻译...
  const elapsed = (Date.now() - startTime) / 1000;
  const speed = (i + 1) / elapsed;
  setTranslationSpeed(speed);
}
```

### 功能3：预估剩余时间

```typescript
const remainingSentences = totalCount - translatedCount;
const estimatedTime = remainingSentences / translationSpeed;

// 显示
{isTranslating && (
  <span>预计剩余 {estimatedTime.toFixed(0)} 秒</span>
)}
```
