## 完整工作流程

```
┌─────────────────────────────────────────────────────────────────┐
│                         用户上传音频                              │
└────────────────────────────┬────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────────┐
│  步骤 1: 音频转录                                                │
│  ─────────────────────────                                      │
│  • 文件验证 (MP3/WAV/AAC, <50MB)                                │
│  • 上传到 /api/voice/transcribe                                 │
│  • SiliconFlow API: 音频 → 中文文本                             │
│  • 状态: "转录中..."                                             │
└────────────────────────────┬────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────────┐
│  步骤 2: 显示转录界面                                            │
│  ──────────────────────────                                     │
│  • 显示 TranscriptionResult 组件                                │
│  • 在片段中显示中文文本                                          │
│  • 显示 "Translation in progress..." 占位符                     │
│  • 状态: "翻译中..." (紫色徽章，动画效果)                        │
└────────────────────────────┬────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────────┐
│  步骤 3: 自动翻译                                                │
│  ──────────────────────────                                     │

│  • 讯飞Lite API: 中文 → 英文                                    │
│  • 温度: 0.3 (稳定)                                             │
│  • 最大令牌数: 4096                                             │
└────────────────────────────┬────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────────┐
│  步骤 4: 更新界面显示翻译                                        │
│  ────────────────────────────────                               │
│  • 将翻译分割为句子                                              │
│  • 与转录片段匹配                                                │
│  • 用真实翻译更新界面                                            │
│  • 状态: "已完成" (绿色徽章)                                     │
└─────────────────────────────────────────────────────────────────┘
```

## 数据流

```
音频文件 (MP3/WAV)
    │
    ├─> [SiliconFlow API]
    │       │
    │       └─> 中文文本: "你好，欢迎使用语音转录系统。这是一个测试。"
    │               │
    │               ├─> 按标点分割: ["你好，欢迎使用语音转录系统", "这是一个测试"]
    │               │
    │               └─> 创建带占位符的片段:
    │                   [
    │                     { originalText: "你好，欢迎使用语音转录系统。",
    │                       translatedText: "Translation in progress..." },
    │                     { originalText: "这是一个测试。",
    │                       translatedText: "Translation in progress..." }
    │                   ]
    │
    └─> [讯飞 API]
            │
            └─> 英文文本: "Hello, welcome to the voice transcription system. This is a test."
                    │
                    ├─> 按标点分割: ["Hello, welcome to the voice transcription system", "This is a test"]
                    │
                    └─> 用翻译更新片段:
                        [
                          { originalText: "你好，欢迎使用语音转录系统。",
                            translatedText: "Hello, welcome to the voice transcription system." },
                          { originalText: "这是一个测试。",
                            translatedText: "This is a test." }
                        ]
```

```
┌─────────────────────────────────────────────────────────────┐

└─────────────────────────────────────────────────────────────┘
```

## API 请求/响应示例

### 转录请求

```bash
POST /api/voice/transcribe
Content-Type: multipart/form-data

file: [二进制音频数据]
```

### 转录响应

```json
{
  "transcription": "你好，欢迎使用语音转录系统。这是一个测试。",
  "duration": 5.2,
  "fileSize": 690747
}
```

### 翻译请求

```bash
POST /api/voice/translate
Content-Type: application/json

{
  "text": "你好，欢迎使用语音转录系统。这是一个测试。",
  "sourceLanguage": "Chinese",
  "targetLanguage": "English"
}
```

### 翻译响应

```json
{
  "translatedText": "Hello, welcome to the voice transcription system. This is a test.",
  "sourceLanguage": "Chinese",
  "targetLanguage": "English",
  "usage": {
    "promptTokens": 45,
    "completionTokens": 18,
    "totalTokens": 63
  }
}
```

## 核心特性

✅ **自动化**: 上传后无需用户额外操作
✅ **快速**: 尽可能并行处理
✅ **可靠**: 优雅的错误处理
✅ **用户友好**: 清晰的状态指示
✅ **双语**: 左右分栏显示
✅ **交互式**: 音频同步、片段高亮
✅ **响应式**: 适配所有屏幕尺寸
