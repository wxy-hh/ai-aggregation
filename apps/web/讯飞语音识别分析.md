## 🎯 讯飞语音听写 API 详情

### 接口要求

**请求协议**：WebSocket (wss://)

**请求地址**：

```
wss://iat-api.xfyun.cn/v2/iat  (推荐)
```

**音频要求**：

- 采样率：16k 或 8k
- 位长：16bit
- 声道：单声道
- 格式：PCM、MP3、SPEEX
- 最长时长：60秒

**支持的语言**：

- 中文普通话 ✅
- 英文 ✅
- 23种方言 ✅
- 多种小语种 ✅

### 核心功能

#### 1. 实时流式识别

```
边上传音频 → 边返回识别结果
```

#### 2. 时间戳支持 ⭐

```json
{
  "data": {
    "result": {
      "ws": [
        {
          "bg": 0, // 起始帧偏移（1帧=10ms）
          "cw": [{ "w": "你好" }]
        }
      ]
    }
  }
}
```

#### 3. 动态修正

- 可以实时修正之前的识别结果
- 提高准确率

#### 4. 标点符号

- 自动添加标点
- 可控制标点位置

### 认证方式

**需要三个参数**：

1. **AppID** - 应用ID
2. **APIKey** - API密钥
3. **APISecret** - API密钥

**获取方式**：

1. 登录讯飞开放平台：https://www.xfyun.cn/
2. 创建应用
3. 添加"语音听写（流式版）"服务
4. 查看认证信息

### 鉴权方式

使用 HMAC-SHA256 签名：

```
authorization = base64(
  api_key="xxx",
  algorithm="hmac-sha256",
  headers="host date request-line",
  signature=base64(hmac_sha256(signature_origin, api_secret))
)
```
