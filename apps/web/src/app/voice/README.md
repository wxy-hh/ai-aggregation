# 语音转写功能

## 功能概述

语音转写页面支持两种模式：

1. **实时录音** - 实时录制并转写语音
2. **上传音频** - 上传本地音频文件进行转写

## 功能特性

### 实时录音模式

- 实时音频波形可视化
- 实时转录文本显示
- 录音控制（开始/暂停）
- 转录文本复制和导出
- 发送至 AI 对话

### 上传音频模式

- 拖拽上传音频文件
- 支持格式：MP3, WAV, AAC
- 文件大小限制：最大 50MB
- 上传进度显示
- 文件预览和管理

## 使用说明

### 切换模式

在页面顶部使用 Tab 切换按钮在"实时录音"和"上传音频"之间切换。

### 上传音频

1. 点击"上传音频"标签
2. 拖拽音频文件到上传区域，或点击"选择文件"按钮
3. 等待文件上传完成
4. 系统自动开始转写

### 实时录音

1. 点击"实时录音"标签
2. 点击录音按钮开始录制
3. 实时查看转录结果
4. 点击暂停按钮停止录制

## API 集成

### SiliconFlow 语音转录 API

**端点：** `POST https://api.siliconflow.cn/v1/audio/transcriptions`

**请求参数：**

- `file` (required) - 音频文件
- `model` (required) - 模型名称
  - `FunAudioLLM/SenseVoiceSmall`
  - `TeleAI/TeleSpeechASR`

**响应格式：**

```json
{
  "text": "转录的文本内容"
}
```
