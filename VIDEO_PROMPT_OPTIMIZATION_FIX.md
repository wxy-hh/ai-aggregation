# 视频提示词智能优化功能修复

## 问题描述

视频生成界面的"智能优化"功能只是简单地在用户输入后面添加固定文本：

```
"8K超清画质，电影级调色，流畅的镜头运动，自然光影效果"
```

这并不是真正的智能优化，无法根据用户的具体输入生成个性化的优化建议。

## 问题原因

原实现代码（`apps/web/src/app/video/_components/config-panel.tsx`）：

```typescript
const handleOptimize = () => {
  if (!prompt.trim()) return;
  setIsOptimizing(true);
  setTimeout(() => {
    setPrompt(prompt + '，8K超清画质，电影级调色，流畅的镜头运动，自然光影效果');
    setIsOptimizing(false);
  }, 1500);
};
```

只是简单的字符串拼接，没有调用AI模型进行真正的优化。

## 解决方案

### 1. 创建提示词优化API

**文件：** `apps/web/src/lib/api/prompt-optimization.ts`

提供客户端调用的API接口：

```typescript
export async function optimizePrompt(options: OptimizePromptOptions): Promise<OptimizePromptResult>;
```

### 2. 创建API路由

**文件：** `apps/web/src/app/api/video/optimize-prompt/route.ts`

使用讯飞Lite模型（免费且快速）进行提示词优化：

**优化规则：**

1. 保留用户原始意图和核心内容
2. 添加视觉细节：光影、色彩、质感、氛围
3. 添加镜头语言：运镜方式、景别、角度
4. 添加技术参数：画质、帧率、后期效果
5. 使用专业术语但保持自然流畅
6. 控制在150字以内，简洁有力

**系统提示词：**

```
你是一个专业的视频生成提示词优化专家。你的任务是将用户简单的描述转换为详细、专业的视频生成提示词。
```

### 3. 更新配置面板组件

**文件：** `apps/web/src/app/video/_components/config-panel.tsx`

新的 `handleOptimize` 函数：

```typescript
const handleOptimize = async () => {
  if (!prompt.trim()) return;

  setIsOptimizing(true);
  try {
    const response = await fetch('/api/video/optimize-prompt', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({
        prompt: prompt.trim(),
        aspectRatio: config.aspectRatio,
        duration: config.duration,
      }),
    });

    if (!response.ok) {
      throw new Error('优化失败');
    }

    const result = await response.json();
    setPrompt(result.optimizedPrompt);
  } catch (error) {
    console.error('Prompt optimization error:', error);
    // 如果优化失败，使用简单的后缀增强作为降级方案
    setPrompt(prompt + '，8K超清画质，电影级调色，流畅的镜头运动，自然光影效果');
  } finally {
    setIsOptimizing(false);
  }
};
```

## 环境变量配置

确保在 `.env.local` 中配置了讯飞API密钥：

```env
XUNFEI_API_PASSWORD=your_api_password
```

**注意：** 讯飞使用的是 `XUNFEI_API_PASSWORD`，不是 API Key。这是讯飞开放平台提供的 API 密码。

### 如何获取讯飞 API 密码

1. 访问 [讯飞开放平台](https://console.xfyun.cn/)
2. 注册/登录账号
3. 创建应用
4. 在应用详情页获取 API 密码（API Password）
5. 将密码配置到环境变量中
