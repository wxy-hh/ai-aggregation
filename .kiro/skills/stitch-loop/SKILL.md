# Stitch 循环技能

这个技能能够从单个提示词生成完整的多页面网站，自动处理文件组织和验证，大幅提升网站开发效率。

## 主要功能

### 1. 多页面生成

- 从一个提示词生成完整的网站结构
- 自动创建多个相关页面
- 保持页面间的设计一致性

### 2. 自动化文件组织

- 智能组织生成的文件结构
- 创建合理的目录层级
- 管理资源文件（图片、样式等）

### 3. 验证功能

- 自动验证生成的代码
- 检查页面间的链接完整性
- 确保设计系统的一致性

## 使用场景

当你需要进行以下工作时，使用此技能：

- 快速搭建完整的网站原型
- 生成多页面的营销网站
- 创建产品展示网站
- 构建内容驱动的网站

## 工作流程

1. **需求分析**：理解网站的整体需求和页面结构
2. **批量生成**：使用 Stitch 生成所有必要的页面
3. **文件组织**：自动整理文件到合适的目录结构
4. **验证检查**：确保所有页面和链接正常工作
5. **优化调整**：根据需要进行微调

## 前置要求

- 已配置 Stitch MCP 服务器
- 明确的网站结构规划
- 了解目标网站的页面类型

## 最佳实践

- 在开始前规划好网站的页面结构
- 提供清晰的网站整体描述
- 明确各页面的功能和内容
- 使用统一的设计系统
- 生成后进行全面的验证测试
