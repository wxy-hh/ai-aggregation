# 语音转录结果 UI 测试指南

## 概述

本指南帮助你测试新实现的语音转录结果展示界面。

## 功能清单

### ✅ 已实现功能

1. **双语对照布局**
   - 左栏显示原文（中文）
   - 右栏显示译文（英文）
   - 两栏同步滚动

2. **视图模式切换**
   - 仅原文模式
   - 仅译文模式
   - 双语对照模式

3. **音频播放器**
   - 播放/暂停控制
   - 进度条显示
   - 当前时间/总时长显示
   - 播放时自动高亮对应片段

4. **片段交互**
   - 点击片段跳转到对应时间点
   - 当前播放片段高亮显示
   - 说话人颜色编码（A=蓝色、B=紫色、C=橙色）

5. **快捷操作**
   - 复制译文到剪贴板
   - 导出按钮（功能待实现）
   - 发送到对话按钮（功能待实现）

## 测试步骤

### 1. 启动开发服务器

```bash
cd apps/web
pnpm dev
```

访问: http://localhost:3000/voice

### 2. 切换到上传模式

1. 点击页面顶部的 "上传音频" 标签
2. 确认界面切换到上传界面

### 3. 上传音频文件

1. 准备一个测试音频文件（MP3、WAV 或 AAC 格式，最大 50MB）
2. 拖拽文件到上传区域，或点击 "选择文件" 按钮
3. 等待上传和转录完成（显示 "转录中..." 状态）

### 4. 测试结果界面

#### 4.1 基本显示

- [ ] 确认界面自动切换到结果展示页面
- [ ] 确认文件名正确显示在顶部
- [ ] 确认语言信息显示（原语言: 中文，目标语言: English）
- [ ] 确认状态显示为 "已完成"

#### 4.2 视图模式切换

- [ ] 点击 "仅原文" 按钮，确认只显示中文文本
- [ ] 点击 "仅译文" 按钮，确认只显示英文文本
- [ ] 点击 "双语对照" 按钮，确认显示左右两栏

#### 4.3 音频播放

- [ ] 点击播放按钮，确认音频开始播放
- [ ] 确认进度条随播放进度移动
- [ ] 确认时间显示正确更新
- [ ] 点击暂停按钮，确认音频暂停

#### 4.4 片段高亮

- [ ] 播放音频时，确认当前片段自动高亮
- [ ] 确认高亮片段有蓝色边框和背景
- [ ] 确认高亮片段左侧有蓝色指示条

#### 4.5 片段交互

- [ ] 点击任意片段，确认音频跳转到对应时间点
- [ ] 确认点击后该片段高亮显示
- [ ] 确认音频从该片段开始播放

#### 4.6 说话人颜色

- [ ] 确认 Speaker A 的头像是蓝色渐变
- [ ] 确认 Speaker B 的头像是紫色渐变
- [ ] 确认 Speaker C 的头像是橙色渐变

#### 4.7 快捷操作

- [ ] 点击 "复制译文" 按钮
- [ ] 粘贴到文本编辑器，确认复制成功
- [ ] 点击 "导出" 按钮，确认显示 "导出功能开发中" 提示
- [ ] 点击 "发送到对话" 按钮，确认显示 "发送到对话功能开发中" 提示

## 已知限制

### 当前使用 Mock 数据

⚠️ **重要**: 当前转录结果界面使用的是 mock 数据，包括：

1. **片段数据**: 5 个预设的对话片段
2. **翻译文本**: 预设的英文翻译
3. **说话人标签**: 预设的 Speaker A/B/C
4. **时间戳**: 预设的时间点

**实际 API 返回的数据**:

- 只有完整的转录文本（没有分段）
- 没有翻译
- 没有说话人识别
- 没有时间戳

### 待实现功能

1. **翻译 API 集成**
   - 需要集成翻译服务（DeepL、Google Translate 等）
   - 在转录完成后自动调用翻译

2. **说话人识别**
   - 需要集成说话人分离技术
   - 自动识别不同说话人

3. **音频分段**
   - 需要实现音频分段算法
   - 生成带时间戳的片段

4. **音频播放器增强**
   - 进度条点击跳转
   - 播放速度控制
   - 音量控制
   - 键盘快捷键

5. **导出功能**
   - 导出为 TXT
   - 导出为 SRT
   - 导出为 PDF
   - 导出为 Word

6. **发送到对话**
   - 集成现有聊天功能
   - 支持发送原文或译文

## 性能测试

### 文件大小测试

- [ ] 测试 1MB 文件
- [ ] 测试 10MB 文件
- [ ] 测试 25MB 文件
- [ ] 测试 50MB 文件（最大限制）

### 响应时间测试

记录以下时间：

1. **上传时间**: 从选择文件到上传完成
2. **转录时间**: 从上传完成到转录完成
3. **界面切换时间**: 从转录完成到结果界面显示
4. **音频加载时间**: 从界面显示到音频可播放

### 预期性能指标

- 上传时间: < 10 秒（50MB）
- 转录时间: < 30 秒
- 界面切换: < 1 秒
- 音频加载: < 2 秒
