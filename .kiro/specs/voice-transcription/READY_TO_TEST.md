## 快速开始

### 1. 启动开发服务器

```bash
cd apps/web
pnpm dev
```

### 2. 访问语音转录页面

打开浏览器访问: http://localhost:3000/voice

### 3. 测试方法

#### 方法 A：快速测试（推荐）

1. 点击"上传音频"标签
2. 点击"测试：直接显示结果界面"按钮
3. 应该立即看到双语对照结果界面

#### 方法 B：真实上传测试

1. 点击"上传音频"标签
2. 准备一个音频文件（MP3、WAV 或 AAC，最大 50MB）
3. 拖拽文件到上传区域，或点击"选择文件"
4. 等待转录完成
5. 应该自动切换到结果界面

## 如果遇到问题

### 1. 页面空白或报错

**检查**:

- 浏览器控制台是否有错误
- Network 标签是否有失败的请求
- 是否正确启动了开发服务器

**解决**:

```bash
# 重启开发服务器
cd apps/web
pnpm dev
```

### 2. 上传后没有反应

**检查**:

- 浏览器控制台的日志
- 是否看到"开始上传文件"日志
- 是否看到"转录结果"日志
- 是否看到"设置 showResult 为 true"日志

**解决**:

- 使用测试按钮验证 UI 是否正常
- 检查 API 请求是否成功
- 查看 Network 标签中的响应内容

### 3. QueryClient 错误

**检查**:

- 是否看到 "No QueryClient set" 错误

**解决**:

- 确认已经添加了 `QueryProvider`
- 确认根布局已经更新
- 重启开发服务器

### 4. 音频无法播放

**检查**:

- 浏览器控制台是否有音频加载错误
- 音频文件格式是否支持
- 音频 URL 是否正确

**解决**:

- 使用测试按钮（会使用预设的音频文件）
- 尝试不同格式的音频文件
- 检查浏览器是否支持该音频格式
