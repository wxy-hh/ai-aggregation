# Stitch 设计技能

这是一个用于 Stitch 设计工作的统一入口点技能。它整合了多个设计相关的功能，帮助你更高效地使用 Stitch MCP 服务器进行 UI/UX 设计。

## 主要功能

### 1. 提示词增强

- 自动添加 UI/UX 相关关键词
- 注入设计氛围和风格描述
- 优化提示词以获得更好的生成结果

### 2. 设计系统合成

- 分析现有设计并生成 `.stitch/DESIGN.md` 文件
- 以自然语义化的语言记录设计系统
- 为后续屏幕生成提供设计上下文

### 3. 高保真屏幕生成

- 通过 Stitch MCP 生成高质量的 UI 屏幕
- 支持屏幕编辑和迭代优化
- 保持设计一致性

## 使用场景

当你需要进行以下工作时，使用此技能：

- 从零开始创建新的 UI 设计
- 优化和增强设计提示词
- 建立和维护设计系统文档
- 生成或编辑应用程序屏幕

## 工作流程

1. **提示词准备**：将用户的设计想法转换为优化的提示词
2. **设计系统建立**：如果需要，创建或更新设计系统文档
3. **屏幕生成**：使用 Stitch MCP 生成高保真的 UI 屏幕
4. **迭代优化**：根据反馈进行编辑和改进

## 前置要求

- 已配置 Stitch MCP 服务器
- 了解基本的 UI/UX 设计概念
- 准备好设计需求和目标

## 最佳实践

- 提供清晰、具体的设计需求
- 在生成前明确设计风格和氛围
- 利用设计系统文档保持一致性
- 迭代式地优化设计结果
