# Remotion 视频技能

这个技能使用 Remotion 框架从 Stitch 项目生成专业的演示视频，通过流畅的动画和过渡效果展示应用的各个屏幕。

## 主要功能

### 1. 视频生成

- 从 Stitch 屏幕生成演示视频
- 自动编排屏幕展示顺序
- 生成高质量的视频输出

### 2. 动画效果

- 流畅的屏幕过渡动画
- 缩放和平移效果
- 淡入淡出转场

### 3. 文字叠加

- 添加说明文字和标题
- 突出显示关键功能
- 自定义文字样式和位置

### 4. 专业展示

- 模拟真实的用户交互
- 展示应用的完整流程
- 适合产品演示和营销

## 使用场景

当你需要进行以下工作时，使用此技能：

- 创建产品演示视频
- 制作应用介绍视频
- 生成营销宣传材料
- 展示设计原型
- 制作用户教程视频

## 工作流程

1. **屏幕准备**：收集 Stitch 项目中的所有屏幕
2. **流程规划**：确定屏幕展示的顺序和逻辑
3. **动画设计**：设计过渡动画和效果
4. **文字编写**：准备说明文字和标题
5. **视频生成**：使用 Remotion 渲染视频
6. **优化调整**：根据需要调整时长和效果

## 前置要求

- Stitch 项目及其屏幕
- Node.js 环境
- Remotion 依赖安装
- 基本的视频制作概念

## 视频特性

- **高质量输出**：支持 1080p 或 4K 分辨率
- **可编程动画**：使用 React 和 CSS 创建动画
- **灵活配置**：自定义时长、帧率、尺寸
- **批量生成**：一次生成多个视频变体
- **格式支持**：输出 MP4、WebM 等格式

## 动画效果示例

- **淡入淡出**：屏幕平滑切换
- **滑动过渡**：从左/右/上/下滑入
- **缩放效果**：放大特定区域
- **聚焦动画**：突出显示关键元素
- **路径动画**：模拟用户操作路径

## 最佳实践

- 保持视频时长适中（1-3 分钟）
- 使用一致的过渡效果
- 文字简洁明了，易于阅读
- 配合背景音乐或旁白
- 突出显示核心功能和价值
- 考虑目标观众的需求
- 测试不同设备上的播放效果
- 优化视频文件大小
